# DramaTune 🎬🎵

DramaTune is a web application that helps creators find and merge the most fitting music with dramatic video scenes using artificial intelligence.

## 🚀 Features

- **Smart Matching:** Upload a video, and get the top 3 music tracks recommended by an AI model (Random Forest).
- **Preview & Sync:** Instantly preview the video with each suggested track—music syncs with video playback.
- **Download:** Download your video merged with your chosen music track.
- **User-Friendly:** Clean, modern React-based UI.

## 🛠️ How It Works

1. **Upload** a dramatic video file.
2. **DramaTune** analyzes the video’s rhythm and suggests three matching music tracks from your collection.
3. **Preview** your video with each track, and select your favorite.
4. **Download** the final merged video.

## 🏗️ Tech Stack

- **Frontend:** React (JavaScript)
- **Backend:** Python (Flask)
- **Machine Learning:** Random Forest (scikit-learn)
- **Video/Audio Analysis:** moviepy, mediapipe, OpenCV
- **Deployment:** (Local machine; see instructions below)
